# ExpoFire #
A simple python script to exploit vulnerable Firebase Database.

# Installation #
## Prerequisited ##
* Python 3.x

## Dependencies ##
* setuptools
* requests

## Installing ##
### Using setup file ###
```
- git clone
- cd ExpoFire
- sudo python3 setup.py install
```
